Calendar Room Booker

Reserve a room with calendar event creation.

Version: V17.1.0.0

Created and maintained by
MyOdoo.pl
assistance and support
kontakt@myodoo.pl

Description

Create a room booking while creating calendar's event.
Booking will be edited accordingly to event.
Working with recurring events!

Usage

Select a room while creating a new calendar event.



After saving the calendar event, new room booking record is created.
All Your's feature changes to the event will also be automatically moved to the booking!

Installation

This module requires Odoos 'calendar' and 'room' to be installed.

Download → Unzip → Move to your addons directory → Update the apps list → Find "Calendar Room Booker" module → Install module
